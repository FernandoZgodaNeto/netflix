Executar

1. Na pasta raiz do projeto, abra o terminal ou CMD e execute os seguintes comandos:

   mvn clean package

E espera terminar a construção do projeto. Isso irá gerar um arquivo JAR com todas as dependências.

2. Depois, execute o seguinte comando para rodar o programa:

   java -cp "target\netflix-1.0-SNAPSHOT-jar-with-dependencies.jar" NetflixAnalyzer "input\netflix_titles.csv" "output"

Pressione ENTER e pronto!

❌ Se houver erro:
   - Verifique se o arquivo netflix_titles.csv existe em:
     input\

   - Verifique se Java está instalado:
     Abra CMD e digite: java -version
